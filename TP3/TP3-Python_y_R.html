  1. R permite la generacion de numeros aleatorios de manera nativa. En este ambiente es un requerimiento esencial para en investigacion cientifica. Entre las herramientas implementadas en el paquete base de R podemos destacar:

  • set.seed(entero): permite establecer la semilla (y el generador).

  • RNGkind(): selecciona el generador.

  • rdistribucin(n,...): genera valores aleatorios de la correspondiente distribucin. Por ejemplo, runif(n, min = 0, max = 1), generara n valores de una uniforme. Se puede acceder al listado completo de las funciones disponibles en el paquete stats mediante el comando ?distributions.

  • sample(): genera muestras aleatorias de variables discretas y permutaciones (se tratar en el Captulo 5).

  • simulate(): genera realizaciones de la respuesta de un modelo ajustado.

    Tambien es posible generar muestras aleatorias utilizando funciones incorporadas como runif(), rnorm(), rpois() y rexp() para distribuciones uniforme, normal, de Poisson y exponencial respectivamente.

d) Los graficos tambien pueden ser generados de manera nativa metiante la funcion hist(). El histograma es el grfico ms habitual para representar datos continuos que representa las frecuencias en las que aparecen las mediciones agrupadas en ciertos intervalos y cuenta cuntas observaciones caen en cada intervalo. Adems, la altura est determinada por el ratio entre la frecuencia y el ancho del intervalo. Con la funcion hist pueden encontrar diferentes opciones grficas. Por ejemplo:

Con border = blue y col = orange, puede cambiar los colores de las barras y de los bordes.

Cuando hay mltiples diagramas, puede utilizar la opcin col=c(darkblue,red, yellow, green, orange, violet).

Con las = c(0, 1,2,3), se seleccionan las orientaciones de los textos: las=0 (paralelo a los ejes, por defecto), las=1 (horizontal), las=2 (perpendicular) y las=3 (vertical).

La opcin horiz=TRUE crea un diagrama horizontal.

Los ttulos de las etiquetas se pueden escribir as: xlab = expression(X[2]), main = , ylab=", names.arg=c(Primer nombre,Segundo nombre").

Los lmites de los ejes se pueden modificar, por ejemplo, as: xlim = c(0, 1), ylim = c(0, 100).

Con la opcin breaks se da un valor con el cual se indica el nmero aproximado de intervalos de clases o un vector cuyos elementos indican los puntos lmites entre las clases o intervalos.

Con las opciones prob=TRUE, freq= F, freq=FALSE o freq=cumsum(F), el grfico se construye con frecuencias relativas.

La funcin abline sirve para aadir una lnea a un grfico. Esta funcin tiene varios parametros con nombre, que sirven para especificar de forma sencilla distintos tipos de lneas:

abline(a, b): lnea de pendiente b y ordenada en el origen a abline(h=y): lnea horizontal abline(v=x): lnea vertical

def calcular_intervalo_confianza(media_muestral, mult_desvio, desvio_estandar, z, n):
